```markdown
# 项目简介

本项目是一款基于Spring Boot的民宿管理系统，旨在提供便捷的民宿管理解决方案。系统采用Vue前端框架，支持管理员和用户两种角色，覆盖民宿管理的多个方面。

## 核心功能

- **管理员功能**：
  - 基础数据库管理
  - 用户管理
  - 轮播图管理
  - 个人中心
  - 房间管理
  - 评论管理
  - 预约管理
  - 公告管理

- **用户功能**：
  - 注册登录
  - 个人中心
  - 首页
  - 房间预定
  - 论坛
  - 公告信息
  - 评论留言
  - 查看房间列表

## 适用场景

- 民宿、客栈等小型住宿场所的管理
- 提供便捷的用户注册和登录体验
- 支持民宿房间的在线预定和评论留言
- 管理员可轻松管理用户信息、房间信息、评论和预约

## 技术特点

- 使用Spring Boot构建，具有高效、稳定的特点
- 前端采用Vue框架，界面友好，易于使用
- 支持多种数据库操作，如Navicat
- 运行环境兼容性强，支持JDK1.8、Node16、Mysql8、Maven3.6
```
